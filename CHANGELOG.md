# User Changelog

All notable user-facing changes to the BB Creation Panel are documented here.

## 1.0.0 - 2026-07-18

### Launch Creation

- Added a responsive token creation flow with name, symbol, logo, description, and validated social links.
- Added launch support for Base, Robinhood, Solana, Ethereum, BSC, and additional networks.
- Added launch plans with clearer selected states, pricing, live completion status, and mobile launch controls.
- Added a live momentum-card preview using the configured token identity, description, and social links.

### DEX Configuration

- Added DEX selection with official Uniswap and PancakeSwap branding.
- Added editable market-cap targets, token-supply presets, token starting-price calculations, and graduation proceeds.
- Added stepped DEX-liquidity controls with health guidance and volatility education.
- Added DEX-tier information, fee guidance, optional token-fee burning, and v3 compatibility handling.

### Fee Builder

- Added configurable creator revenue, buybacks and burns, liquidity, treasury, custom routes, and Rewards Basket routing.
- Added Rewards, Creator, Balanced, Mixed, and CTO presets with reset support.
- Added a Rewards Basket spanning tokens, stocks, and ETFs with search, category filters, quick picks, select-all, reset, and selected-only viewing.
- Added rotating and all-at-once basket distribution modes, custom asset allocations, and holder eligibility thresholds.
- Added automatic fee distribution triggers and clear route validation, medium-fee, and high-fee guidance.
- Added experimental protections for tiered and dynamic fees, wallet limits, cooldowns, launch controls, snipe protection, and MEV protection.

### Launch Options

- Added initial-buy controls, estimated token purchases, market-cap impact, and multi-wallet distribution.
- Added LBP controls for sale timing, fee configuration, buyer limits, soft caps, trade delays, and whitelist access.
- Added wallet funding checks for creation costs, plan fees, gas, and initial buys, including Base CDP gas sponsorship behavior.

### Experience And Accessibility

- Added smooth expansion, centering, filtering, and mobile sheet transitions throughout the creation flow.
- Added full-screen mobile distribution and whitelist experiences with compact responsive controls.
- Improved helper text placement, keyboard focus, validation messages, hover states, and touch targets.
- Refined contrast, spacing, typography, route colors, icons, and applied-settings summaries across desktop and mobile.
